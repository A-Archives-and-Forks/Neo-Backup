Backup modes: An app can be backed up with only its apk, only its data or both. An app not currently installed can only be restored if the apk is present.

Borrar la copia de seguridad: Elimina los archivos de la copia de seguridad específicada. Mientras que Borrar las copias de seguridad Elimina todas las copias de seguridad de la aplicación elegida.

Desinstalación: Algo más agresiva que una desinstalación normal. Además de realizar una desinstalación normal a través de los comandos de Android, la desinstalación desde OAndBackupX elimina los archivos que la aplicación pueda haber dejado en /data/app-lib/.

Habilitar / deshabilitar: Utiliza el script de Android " pm " para habilitar o deshabilitar una aplicación. Desactivar una aplicación la elimina de la interfaz de usuario normal sin desinstalarla. Se puede utilizar para habilitar o deshabilitar una aplicación para varios usuarios a la vez (si el dispositivo tiene varios usuarios). Los usuarios se identifican con un ID, por ejemplo, 0 es el primer usuario (propietario) 10 suele ser el perfil de trabajo.

Usuarios Múltiples: El modo multiusuario es todavía algo experimental pero debería funcionar. Habilitarlo y deshabilitarlo sólo funciona después de una instalación inicial (no necesariamente desde OAndBackupx) o restaurando la aplicación.

Aplicaciones System-less: el respaldo y la restauración de las aplicaciones System-less como los módulos de Magisk , no es compatible y el intento podría conducir a un fallo del sistema y obligar a formatear y flashear de nuevo.

Navegación: La barra de navegación tiene "Ajustes", "Inicio", "Respaldo por lotes", "Restauración por lotes" y "Programar respaldos" (en ese orden). El número de aplicaciones actualizadas después de su último respaldo se presentará como una notificación en el botón "Inicio".

Aplicaciones que utilizan almacén de claves: Android tiene un almacenamiento cifrado para los datos sensibles. Las aplicaciones que hacen uso de esto para almacenar datos de acceso o algunas claves para su funcionamiento, no pueden ser restauradas con éxito. Dos ejemplos de estas aplicaciones son Signal y Element(Riot).
